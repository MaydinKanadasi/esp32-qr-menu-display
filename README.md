# ESP32 QR Menu Display

## Overview
This project is a prototype of a restaurant table display device built with an ESP32 and an ILI9341 TFT display. The device shows a QR code that customers can scan to access a digital menu.

## Hardware
- ESP32 DevKit V1
- ILI9341 TFT Display

## Features
- Restaurant name display
- Table number display
- QR code rendering on TFT screen
- Simple prototype for a QR-based restaurant menu system
